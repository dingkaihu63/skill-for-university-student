# 四维AIGC检测框架

本框架用于对思政论文文本进行AIGC（AI生成内容）概率评估，基于文本表层特征与生成模型统计规律进行概率推断。

## 检测维度总览

| 维度 | 检测目标 | 权重 |
|------|----------|------|
| 语言指纹 | AI生成文本的统计性语言特征 | 30% |
| 结构逻辑 | 文本组织模式的机械性与有机性 | 25% |
| 内容实质 | 信息深度、案例真实性与认知轨迹 | 25% |
| 学科特性 | 思政学科特异性特征与AI误判风险 | 20% |

---

## 一、语言指纹维度

### 1.1 句法均匀度检测

**检测指标**：
- 长短句比例是否接近机械均衡（自然文本长短句比例约3:2，AI文本趋向2:1或更均匀）
- 段落字数是否集中于固定区间（AI文本常见200-300字/段的窄幅波动）
- 连续3句以上是否重复相同主语或句式结构

**评分标准**：
- 0-2分：句法自然起伏，长短句交替有机，无明显规律性
- 3-5分：存在局部句式重复，但整体尚可
- 6-8分：句法高度均匀，段落字数集中，典型AI特征
- 9-10分：完全模板化句法，无可读性变化

### 1.2 AI安全过渡词检测

**高频AI过渡词库**（出现3次以上即触发预警）：
- 值得注意的是、综上所述、随着发展、不容忽视、显而易见
- 毋庸置疑、由此可见、总而言之、换言之、与此同时
- 不可否认、众所周知、事实上、一般来说、在某种意义上

**学科内合法承接表达**（不触发预警）：
- 该论述启示我们、结合当前青年群体特征、从理论逻辑延伸至现实关切
- 在此语境下、回溯历史脉络、立足于新时代坐标

### 1.3 语态与主体检测

**检测指标**：
- 被动语态/无主句占比（AI文本常超50%，自然文本应低于30%）
- 是否存在明确的认知主体（"笔者""本调研小组""新时代大学生"等）
- 主语是否频繁缺失或使用泛化代词

### 1.4 术语使用检测

**检测指标**：
- 标准政治与理论术语是否精准使用（非机械堆砌）
- 术语是否存在语境错配
- 理论引用是否与最新权威表述一致（如党的二十大报告、统编教材原文）

## 二、结构逻辑维度

### 2.1 推进模式检测

**AI典型结构**（高风险）：
- 背景→理论→意义→总结（机械分层）
- 提出问题→分析问题→解决问题（三段论套用）

**自然有机结构**（低风险）：
- 理论认知→现实观察→价值辨析→行动指向（有机递进）
- 问题意识→理论回应→实践检验→价值升华

### 2.2 论证闭环检测

**检测指标**：
- 每段是否构建"理论命题-现实映射-青年回应"三角逻辑链
- 是否存在纯政策复述或理论罗列（无个人回应的段落）
- 论证是否形成闭环而非断裂

### 2.3 批判性表达检测

**AI缺失特征**（增加AI概率判定）：
- 关键结论前无设问、边界限定或谨慎表述
- 使用绝对化断言（"必然""一定""毫无疑问"）
- 无任何"初步印证""尚需验证"等留白表达

### 2.4 辩证真实性检测

**伪辩证模式**（高风险）：
- "一方面……另一方面……"但无实质张力分析
- "虽然……但是……"仅为形式转折，无深层价值权衡
- "既……又……"并列但无矛盾化解路径

**真实辩证模式**（低风险）：
- 具价值权衡或矛盾化解路径的论述
- 承认张力存在并尝试回应
- 在对立中寻找统一的具体路径

## 三、内容实质维度

### 3.1 认知轨迹检测

**AI缺失特征**：
- 无思想演进路径标注
- 缺乏"初识→深入→内化"的认知发展过程
- 无个人理解视角的介入

**自然文本特征**：
- 显性标注思想演进（如"初识该理论时仅停留在概念层面，经结合XX现实案例研读后，逐步认识到其方法论价值"）
- 体现"真学-真懂-真信"过程

### 3.2 案例颗粒度检测

**AI泛化表述**（高风险）：
- "参加活动深受启发"
- "社会现象值得关注"
- "通过实践深刻认识到"
- "在调研中发现重要问题"

**自然具体表述**（低风险）：
- 包含时间、地点、调研对象、具体行为
- 包含数据/访谈摘录、个人反思
- 符合大学生身份与能力边界

### 3.3 理论内化检测

**AI特征**：
- 脱离原文语境的空泛发挥
- 理论引用无个人理解视角
- 无现实对照分析

**自然特征**：
- 理论引用附带个人理解视角
- 存在现实对照分析
- 体现"价值认同"的真实思想演进过程

---

## 四、思政学科特异性维度

### 4.1 时代语境契合度

**检测指标**：
- 政策表述是否对齐最新权威文献
- 理论框架是否使用过时表述
- 是否存在与当前政策方向不一致的表述

### 4.2 立场一致性

**检测指标**：
- 价值导向是否清晰稳定
- 是否存在逻辑跳跃或立场漂移
- 政治立场是否前后一致

### 4.3 去主体化风险

**检测指标**：
- 全文是否缺乏青年视角的真实介入
- 是否沦为"理论复读机"或"政策宣讲稿"
- 是否有大学生主体的真实思考和感受

---

## 检测输出格式

对文本进行检测后，必须严格按以下结构返回评估报告：

### 综合评估结论

```
AIGC概率区间：[低/中/高]（对应0-20%/21-50%/51-80%）
检测置信度：[高/中/低]（基于文本长度、特征一致性、学科适配度综合判定）
核心判定依据：[1-3条最显著特征]
```

### 四维特征诊断

```
语言指纹：[评分/关键证据摘录/原文行号或段落标识]
结构逻辑：[评分/关键证据摘录/逻辑断裂点定位]
内容实质：[评分/案例或理论内化缺陷说明]
学科特性：[规范表述说明/AI误判风险预警]
```

### 高风险段落定位

```
[段落编号/首句]：[具体AI指纹解析，如"连续使用模板化连接词+无主句占比78%+案例无时空坐标"]
[段落编号/首句]：[具体分析]
```

### 人工复核建议

```
需重点核查的理论引用或政策表述：[列出]
建议补充的个人认知/实践细节：[指出缺失类型]
逻辑重构方向：[提供1-2条具体修改路径]
```

### 检测局限性声明

本评估基于文本表层特征与生成模型统计规律进行概率推断，无法替代学术不端调查与人工深度审阅。思政论文的规范性政治语言易与AI安全词库重叠，可能导致假阳性；高度仿写的AI文本亦可能通过表面合规检测。最终学术责任由作者与指导教师共同承担。
